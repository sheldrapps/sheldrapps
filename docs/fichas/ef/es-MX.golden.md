# Reparar EPUB - es-MX Golden Play Store Listing

## Purpose

Crear una ficha de Play Store clara, simple y convincente para una app enfocada en una tarea concreta:

**detectar problemas comunes en archivos EPUB, distinguir cuando un archivo no es un EPUB real, reparar lo compatible y guardar una copia nueva del resultado.**

La ficha no debe presentar la app como lector de ebooks, editor EPUB completo, convertidor general, herramienta para quitar DRM, servicio en la nube ni app para descargar libros.

La idea principal debe sentirse así:

**Tu EPUB tiene un problema. La app te ayuda a saber si realmente es un EPUB, revisar qué falló, reparar lo compatible y guardar una copia nueva sin tocar el original.**

## Long Description

Reparar EPUB te ayuda a revisar archivos EPUB con problemas y guardar una copia corregida cuando la reparación es compatible.

Si un EPUB no abre bien, falla al importarse o presenta errores internos, la app te permite analizarlo de forma clara antes de intentar repararlo.

Selecciona un archivo desde tu dispositivo y la app revisa su estructura interna para detectar problemas comunes. También puede identificar cuando un archivo tiene extensión .epub pero en realidad no corresponde a un EPUB válido.

Con Reparar EPUB puedes:

* Seleccionar archivos EPUB desde tu dispositivo
* Detectar si el archivo es realmente un EPUB válido
* Revisar problemas comunes de estructura
* Analizar elementos como mimetype, container.xml, OPF, manifest y spine
* Detectar referencias dañadas o archivos internos faltantes
* Reparar problemas compatibles
* Guardar una copia nueva del EPUB reparado
* Mantener el archivo original sin cambios
* Ver archivos reparados o importados en My EPUBs
* Cambiar idioma y tema desde ajustes
* Quitar anuncios con una opción de mejora

La app está diseñada para mantener el flujo simple: elegir un archivo, entender el problema, reparar lo compatible y guardar el resultado.

Cuando un archivo no es un EPUB real, la app lo indica claramente para evitar que pierdas tiempo intentando repararlo.

Todo el procesamiento del archivo se realiza en tu dispositivo.

Sin cuentas.
Sin subir archivos.
Con control local sobre tus EPUBs.

Algunas reparaciones gratuitas pueden requerir un anuncio recompensado. Si el anuncio no carga, la app muestra una alternativa para mantener el flujo claro.

Reparar EPUB es útil para lectores, estudiantes, coleccionistas y personas que organizan su biblioteca digital y quieren recuperar archivos EPUB con problemas sin depender de herramientas complicadas.

La app se enfoca en una tarea concreta: **diagnosticar, reparar lo compatible y guardar una copia nueva de forma segura**.
