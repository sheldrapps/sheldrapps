## Long Description

Text:
일반적인 EPUB 문제를 확인하고 복구된 사본을 저장하세요.

EPUB Fixer는 제대로 열리지 않거나, 내부 참조가 깨져 있거나, 구조적 문제로 인해 예상대로 동작하지 않는 EPUB 파일을 검사하는 데 도움을 줍니다.

기기에서 EPUB를 선택하고 빠른 진단을 실행하세요. 앱은 지원되는 문제를 명확하게 식별하고, 고칠 수 있는 부분을 복구한 뒤, 결과를 새 EPUB 파일로 저장할 수 있게 합니다.

EPUB Fixer로 할 수 있는 일:

• 일반적인 EPUB 구조 문제 진단
• 누락되었거나 유효하지 않은 EPUB package 파일 및 참조 탐지
• 지원되는 manifest 및 spine 문제 복구
• 무엇이 발견되고 무엇이 복구되었는지 검토
• 복구된 EPUB를 새 사본으로 저장
• 원본 파일을 변경하지 않고 유지

모든 처리는 기기에서 이루어집니다.

계정 없음. 업로드 없음. 클라우드 변환 없음.

EPUB Fixer는 복잡한 데스크톱 소프트웨어 없이 EPUB 문제를 쉽게 해결하고 싶은 독자, 수집가, 디지털 라이브러리 관리자에게 적합합니다.

## Notes / Assumptions

* 리스트는 앱이 실제로 지원하는 복구만 설명한다.
* 진단과 복구 결과 스크린샷은 단순 파일 유틸리티가 아님을 보여주기 때문에 필수다.
* manifest, spine, package structure는 보조적으로만 사용한다.
* 시각적 흐름은 항상 복구: EPUB 문제 발견 → EPUB 복구.
* 가능하면 원본 EPUB가 보존된 상태를 보여준다.
* 계정 없음, 업로드 없음, 기기 내 처리는 중요한 신뢰 증거다.
