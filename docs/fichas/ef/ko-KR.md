## Long Description

Text:
일반적인 EPUB 문제를 복구하고 수리된 복사본을 저장합니다.

EPUB Fixer는 제대로 열리지 않는 EPUB, 내부 참조가 깨진 EPUB, 정상 동작을 방해하는 구조 문제를 가진 EPUB을 확인하는 데 도움을 줍니다.

기기에서 EPUB를 선택하고 해당 파일에 대한 안전한 세션을 준비하세요. 앱은 mimetype, container.xml, OPF 데이터, manifest 항목, spine 참조 같은 중요한 EPUB 구조 요소를 확인합니다.

지원되는 문제가 발견되면 EPUB Fixer는 깨진 참조를 수정하고, 가능하면 EPUB 구조를 재구성한 뒤, 결과를 새 EPUB 파일로 저장하는 복구 흐름으로 안내합니다.

EPUB Fixer에서 할 수 있는 일:

• 기기에서 EPUB 파일 선택
• 각 파일에 대한 안전한 세션 준비
• 일반적인 EPUB 구조 문제 진단
• mimetype, container.xml, OPF, manifest, spine 확인
• 깨진 참조와 누락된 패키지 파일 감지
• 지원되는 EPUB 구조 문제 복구
• 가능하면 EPUB 재구성
• 수리된 EPUB를 새 복사본으로 저장
• 원본 파일은 그대로 유지
• My EPUBs에서 수리 또는 가져온 파일 확인
• 언어와 테마 조정
• 업그레이드 옵션으로 광고 제거

EPUB Fixer는 .epub 확장자를 사용하지만 실제 EPUB가 아닌 문제 파일도 명확하게 처리합니다.

모든 처리는 기기에서 이루어집니다.

계정 없음.
업로드 없음.
클라우드 변환 없음.

일부 무료 수리는 리워드 광고 시청이 필요할 수 있습니다. 광고가 로드되지 않으면 앱이 대체 흐름을 제공하여 다음 단계가 분명하게 보이도록 합니다.

EPUB Fixer는 복잡한 데스크톱 소프트웨어 없이 EPUB 문제를 해결하고 싶은 독자, 수집가, 학생, 디지털 서가 사용자에게 적합합니다.

앱은 하나의 실용적인 일에 집중합니다: 지원되는 EPUB 수리를 진단하고, 복구하고, 안전하게 저장하는 것.

## Notes / Assumptions

* 문구는 실제로 지원되는 복구만 설명합니다.
* 진단 화면과 결과 화면은 이 제품이 단순한 파일 도구가 아니라는 것을 증명하므로 필수입니다.
* manifest, spine, package structure 같은 기술 용어는 보조적으로만 사용하고, 제목을 지배하지 않게 하세요.
* 시각적 스토리는 항상 복원이어야 합니다: 문제 EPUB 발견 -> EPUB 복구.
* 가능하다면 원본 EPUB이 그대로 보존된 모습도 보여주세요.
* 계정 없음, 업로드 없음, 기기 내 처리, My EPUBs는 중요한 신뢰 신호입니다.
