## Long Description

Text:
检查常见 EPUB 问题并保存修复副本。

EPUB Fixer 可帮助你检查无法正常打开、内部引用损坏或存在结构问题而影响使用的 EPUB 文件。

从设备中选择一个 EPUB 并运行快速诊断。应用会清晰识别受支持的问题，修复它能修复的部分，并让你将结果保存为新的 EPUB 文件。

使用 EPUB Fixer，你可以：

• 诊断常见 EPUB 结构问题
• 检测缺失或无效的 EPUB package 文件和引用
• 修复受支持的 manifest 和 spine 问题
• 查看发现了什么、修复了什么
• 将修复后的 EPUB 保存为新副本
• 保持原始文件不变

所有处理都在设备本地完成。

无需账号。无需上传。不进行云端转换。

EPUB Fixer 适合读者、收藏者以及任何需要简单方式排查 EPUB 问题、又不想使用复杂桌面软件管理数字图书馆的人。
