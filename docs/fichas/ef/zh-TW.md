## Long Description

Text:
檢查常見 EPUB 問題並儲存修復副本。

EPUB Fixer 可協助你檢查無法正常開啟、內部參照損壞或存在結構問題而影響使用的 EPUB 檔案。

從裝置中選擇一個 EPUB 並執行快速診斷。應用程式會清楚辨識受支援的問題，修復它能修復的部分，並讓你將結果儲存為新的 EPUB 檔案。

使用 EPUB Fixer，你可以：

• 診斷常見 EPUB 結構問題
• 偵測遺失或無效的 EPUB package 檔案與參照
• 修復受支援的 manifest 和 spine 問題
• 查看發現了什麼、修復了什麼
• 將修復後的 EPUB 儲存為新副本
• 保持原始檔案不變

所有處理都在裝置上完成。

無需帳號。無需上傳。不進行雲端轉換。

EPUB Fixer 適合讀者、收藏者，以及任何需要簡單方式排查 EPUB 問題、又不想使用複雜桌面軟體管理數位圖書館的人。
