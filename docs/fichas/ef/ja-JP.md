## Long Description

Text:
EPUBの一般的な問題を確認し、修復済みコピーを保存します。

EPUB Fixer は、正しく開かない EPUB ファイル、壊れた内部参照を含むファイル、または期待どおりに動作しない構造上の問題を調べるのに役立ちます。

端末から EPUB を選び、素早い診断を実行してください。アプリは対応可能な問題を明確に識別し、修復できるものを修復し、結果を新しい EPUB ファイルとして保存できるようにします。

EPUB Fixer でできること:

• 一般的な EPUB 構造の問題を診断する
• 欠落または無効な EPUB パッケージファイルと参照を検出する
• 対応可能な manifest と spine の問題を修復する
• 見つかった内容と修復内容を確認する
• 修復済み EPUB を新しいコピーとして保存する
• 元のファイルをそのまま保持する

すべての処理は端末上で行われます。

アカウント不要。アップロード不要。クラウド変換なし。

EPUB Fixer は、複雑なデスクトップソフトを使わずに EPUB を簡単にトラブルシュートしたい読者、コレクター、デジタルライブラリ管理者のためのツールです。
