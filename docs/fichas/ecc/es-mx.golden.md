# Cambiar portada a EPUB - es Golden Play Store Listing

## Purpose

Position ECC for the Spanish-speaking market as a fast, simple, and private EPUB utility for readers who want to replace, fix, or personalize the cover of an EPUB file without using complicated software.

The listing should sell the outcome first:

Cambiar la portada de cualquier EPUB en segundos.

The app should feel direct, useful, local, private, and easy to understand. The user should immediately understand that they can select an EPUB file, choose a new image, adjust the framing, preview the result, and save a new copy of the book with the new cover.

The strongest conversion angle is not “editor de ebooks,” “lector EPUB,” “generador de portadas con IA,” or “organizador de biblioteca.” It is:

Tengo un EPUB con una portada fea, incorrecta, vacía o poco atractiva, y quiero cambiarla rápido.

ECC should be presented as a focused EPUB cover changer. It should not be positioned as a reading app, book marketplace, EPUB text editor, DRM tool, converter, or online service.

## Market Angle Decision

Market angle selected:
Cambio rápido de portada EPUB con edición local, privacidad y guardado seguro como copia.

Why this angle should convert in this locale:
Spanish-speaking users are likely looking for a practical fix: a book has the wrong cover, no cover, a basic placeholder, or a cover that makes the library look messy. The listing should not feel technical or abstract. It should show the result clearly: portada mala/simple → portada atractiva.

The strongest trust proof is that the book stays on the device, no account is needed, and the app saves a copy with the new cover.

Primary user desire:
Cambiar una portada incorrecta, simple, fea o ausente por una imagen elegida por el usuario.

Secondary trust proof:
Proceso local, sin cuentas, sin subir archivos, ajuste preciso, vista previa y guardado de una nueva copia.

What should NOT be the first screenshot:
A settings screen, a technical EPUB details screen, an empty library, a generic image editor, or a privacy-only message. The first screenshot must show the problem and the solution immediately.

## Regional Conversion Strategy

Primary user desire:
Arreglar o personalizar portadas EPUB para que la biblioteca digital se vea más limpia, ordenada y agradable.

Best use cases to show:
Portada simple reemplazada por una portada visual atractiva, portada incorrecta corregida, imagen personal adaptada como portada, vista de biblioteca con la portada actualizada, editor mostrando recorte/encuadre.

What to avoid:
Lenguaje de Kindle/e-reader cover, compatibilidad por modelos Kindle, portadas PDF, IA generativa, DRM, descarga de libros, libros gratis, marketplace, lectura dentro de la app o edición de texto EPUB.

Search/ASO terms to include naturally:
Cambiar portada EPUB, editar portada EPUB, reemplazar portada EPUB, editor de portada EPUB, portada de ebook, corregir portada, biblioteca digital, personalizar ebooks, archivo EPUB.

This makes the app feel more trustworthy, modern, and practical without losing the direct utility style that already works.

The warm accent should mainly come from the “después” cover artwork. The UI/background should stay restrained so the transformation remains the visual protagonist.

General visual rules:
Use a dark, practical, high-contrast visual system. The design should feel simple, reliable, and focused.

The strongest visual language is transformation:
portada simple o incorrecta → portada final atractiva.

Typography should stay bold, readable, and direct. Spanish headlines can become longer than English, so line breaks must be controlled carefully to avoid cramped layouts.

* exceso de iconos decorativos

* This golden ficha uses the current high-converting ECC visual direction: dark background, bold white headline, clean gray-blue subline, and clear before/after cover transformation.
