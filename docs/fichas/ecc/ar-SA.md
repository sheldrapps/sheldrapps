Reasoning:
For ar-SA, the visual direction should feel premium, private, calm, and reliable. A dark charcoal base with subtle bronze/gold accents works better than aggressive red because it feels more polished and trust-oriented.

The accent should be used only for controlled emphasis, small proof points, dividers, arrows, or subtle highlights. It should not become a large glow behind the headline.

General visual rules:
Use a clean RTL-aware layout. Keep text areas dark, matte, and readable. The visual transformation must remain the main focus: غلاف ضعيف أو غير مناسب → غلاف جديد واضح وجذاب.

Avoid bright panels behind text, heavy decoration, saturated red glows, dense patterns, AI-like sparkle effects, or cloud/upload symbols.
