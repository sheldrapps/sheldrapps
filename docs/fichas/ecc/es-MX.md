## Purpose

Crear una ficha que comunique con claridad que la app permite cambiar la portada de un archivo EPUB, ajustar la imagen, revisar el resultado y guardar una copia nueva del libro.

La ficha debe sentirse directa, práctica, confiable y fácil de entender. El usuario debe captar de inmediato que puede corregir una portada fea, incorrecta, incompleta o que simplemente no le convence.

## Market Angle Decision

Market angle selected:
Cambiar o corregir portadas EPUB rápido, con procesamiento local y guardado como copia nueva.

Why this angle should convert in this locale:
Para es-MX, el mensaje debe ser claro y práctico. El usuario no necesita una explicación técnica sobre EPUB; necesita ver que puede arreglar una portada que no le gusta o no corresponde al libro.

La promesa más fuerte es: “elige una imagen, ajusta la portada y guarda una copia nueva sin subir tus archivos”.

Primary user desire:
Cambiar una portada EPUB fea, incorrecta, incompleta o poco atractiva por una imagen elegida por el usuario.

Secondary trust proof:
Procesamiento en el dispositivo, sin cuentas, sin subir archivos, vista previa antes de guardar y creación de una copia nueva.

What should NOT be the first screenshot:
Ajustes, biblioteca vacía, detalles técnicos del archivo, editor genérico de imágenes o una pantalla de privacidad sin mostrar primero la transformación de portada.

## Visual System

Palette:

* secondary background: #24202A

* optional support accent: #F08A5D
* headline color: #FFFFFF
* subline color: #F0D6CF

Reasoning:
Para es-MX, la ficha puede usar una base oscura y confiable con un acento cálido moderado. Esto hace que la app se sienta práctica, cercana y visualmente atractiva sin perder seriedad.

El acento coral/naranja debe usarse como énfasis controlado: líneas finas, indicadores antes/después, pequeños puntos de confianza o luz sutil cerca de la portada final. No debe dominar el fondo ni competir con la imagen de portada.

General visual rules:
Mantener el área de texto oscura, mate y legible. La transformación debe ser protagonista: portada débil o incorrecta → portada nueva más atractiva.

Evitar brillos fuertes detrás del texto, fondos demasiado naranjas, elementos decorativos excesivos, señales de IA, iconos de nube/subida, visuales de PDF, visuales de Kindle o cualquier composición que parezca editor genérico de imágenes.
