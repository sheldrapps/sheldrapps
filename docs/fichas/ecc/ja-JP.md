## Long Description

Text:
EPUBファイルの表紙を、好きな画像にすばやく変更できます。

EPUBを選び、新しい画像を追加し、切り抜きや位置を調整してから、完成イメージを確認できます。

このアプリでできること:

• EPUBの元の表紙を差し替え
• 端末内の画像を新しい表紙に使用
• 画像の切り抜き、位置合わせ、調整
• 保存前に表紙をプレビュー
• 新しいEPUBコピーとして保存

処理はすべて端末内で行われます。
アカウント不要。
アップロード不要。

間違った表紙、表示されない表紙、地味な表紙、好みに合わない表紙を整えたいときに便利です。

電子書籍ライブラリを見やすく整理したい方や、EPUBの表紙だけを手軽に変更したい方に向いています。

このアプリは電子書籍リーダー、書籍ストア、完全なEPUB編集ツールではありません。EPUB表紙の変更に特化したシンプルなツールです。

Reasoning:

General visual rules:
