| zh-TW | Sí | 在裝置上處理。無需帳號。不需上傳。 | 快速更換 EPUB 封面 | 避免看起來像一般圖片編輯器或雲端服務 |

Conclusión:
- Todas las variantes mantienen el mismo producto real: reemplazar la portada de un archivo EPUB con una imagen elegida por el usuario.
- La jerarquía de conversión queda alineada como: problema claro → cambio rápido → control de portada → privacidad local.
- No se introdujeron promesas de IA, generación automática, plantillas, DRM, edición completa de EPUB ni afiliación oficial con plataformas de lectura.
- La prueba de confianza principal se mantiene consistente en todos los locales: procesamiento en el dispositivo, sin cuentas y sin subir archivos.
- El primer impacto visual debe mostrar una transformación antes/después, no una pantalla técnica ni una lista de archivos.
- El riesgo principal es que la app parezca un editor de imágenes genérico; por eso cada locale debe abrir con el resultado: cambiar o corregir una portada EPUB.
