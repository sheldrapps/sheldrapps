# Cover Creator for Kindle - es-MX Golden Play Store Listing

## Propósito

Posicionar CCFK para México como una app rápida, clara y Kindle-first para lectores que quieren convertir cualquier imagen que ya tienen en una portada personalizada para Kindle o e-reader.

La ficha debe vender primero el resultado: convertir una imagen en una portada para Kindle en segundos.

La app debe sentirse simple, útil, visual y confiable. El usuario debe entender de inmediato que puede elegir una imagen, seleccionar su modelo de Kindle o e-reader compatible, ajustar el encuadre, revisar el resultado y guardar una portada lista para usar.

El ángulo de conversión más fuerte no es “editor de diseño”, “generador con IA” ni “galería estética”. Es:

Tengo una imagen que me gusta y quiero que quede bien como portada en mi Kindle o e-reader.

Aunque el nombre y el gancho principal de ASO son Kindle-first, la ficha también debe aclarar que la app soporta otras marcas de e-reader como Kobo, Nook, PocketBook y Tolino cuando sea correcto.

## Decisión de ángulo de mercado

Ángulo de mercado seleccionado:
Creación rápida de portadas para Kindle/e-reader desde cualquier imagen existente.

Por qué este ángulo debería convertir en este locale:
En México, la promesa debe entenderse rápido y sin rodeos: imagen → portada para Kindle. El usuario no necesita leer una explicación técnica para captar el valor. La app debe abrir con el resultado práctico y después respaldarlo con recorte, vista previa, selección de modelo, ajuste para tinta electrónica y compatibilidad con otros e-readers.

Deseo principal del usuario:
Convertir una imagen favorita en portada para Kindle sin adivinar tamaños ni usar programas complicados.

Prueba secundaria de confianza:
Tamaño según modelo, recorte, vista previa, ajuste para tinta electrónica y soporte para Kindle, Kobo, Nook, PocketBook, Tolino y más.

Qué NO debe ser la primera captura:
Pantalla de ajustes, selector de modelo, tabla de compatibilidad, editor genérico o una promesa estética amplia sin mostrar el resultado final.

Riesgo de empezar por funciones:
Si la ficha empieza con selección de modelo o detalles técnicos, puede sentirse fría. Si empieza solo con estética, puede no quedar claro qué hace la app. La primera impresión debe mostrar la transformación: una imagen convertida en portada para Kindle.

Deseo principal del usuario:
Crear una portada que haga que su Kindle o e-reader se vea más personal, bonito y listo para usar.

Reglas visuales generales:

Evitar brillos intensos detrás del headline, paneles blancos detrás del texto, fondos saturados, UI diminuta, exceso de decoración o visuales que parezcan edición de EPUB.

## Notas / Supuestos

- Esta golden ficha aplica la estructura de conversión de ECC a CCFK sin cambiar el producto.
- CCFK no cambia portadas de EPUB. Crea/adapta portadas para Kindle y e-readers desde imágenes que el usuario ya tiene.
- La app no promete generar imágenes, arte con IA, plantillas ni ilustraciones.
- Los nombres de marcas se usan solo de forma descriptiva. La ficha no debe insinuar afiliación oficial con Amazon, Kindle, Kobo ni otras marcas.
- La compatibilidad debe expresarse como familias o modelos soportados, salvo que la cobertura exacta esté verificada en el proyecto actual.
- La secuencia de conversión más fuerte es: resultado imagen-a-portada → flujo simple → ajuste por modelo → vista previa/guardar → compatibilidad.
- Esta golden ficha es una referencia de calidad para generación con Codex, no una plantilla literal para copiar en todos los locales.
