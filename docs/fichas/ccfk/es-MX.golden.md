# Crear Portadas para Kindle - es-MX Golden Play Store Listing

## Purpose

Posicionar CCFK para México como una app Kindle-first de personalización visual, pensada para usuarios que quieren que su Kindle o e-reader se sienta más suyo, más bonito y más personal.

La ficha debe vender primero el deseo: convertir una foto querida, una mascota, un recuerdo o una imagen favorita en una portada personalizada para Kindle.

Las funciones técnicas como selección de modelo, recorte, vista previa y exportación deben aparecer después como prueba de confianza.

Aunque el nombre y el gancho ASO principal usan Kindle, la ficha debe aclarar que también soporta otros e-readers como Kobo, Nook, PocketBook y Tolino cuando sea correcto.

## Long Description

Text:
Haz que tu Kindle se sienta más tuyo.

Crear Portadas para Kindle te ayuda a convertir imágenes que ya tienes en portadas personalizadas para tu Kindle y otros e-readers compatibles. Usa una foto favorita, una imagen de tu mascota, un recuerdo de viaje, una ilustración, arte digital o cualquier visual que quieras ver en tu pantalla de lectura.

No necesitas adivinar tamaños ni pelearte con ajustes raros. Elige tu imagen, selecciona el modelo de tu dispositivo, ajusta el encuadre, revisa la vista previa y crea una portada lista para usar.

Úsala para:
- fotos de tu mascota
- recuerdos personales o de viaje
- imágenes aesthetic para tu Kindle
- arte, ilustraciones o diseños que ya tengas
- portadas personalizadas para Kindle, Kobo, Nook, PocketBook, Tolino y más

La idea es simple: tomar una imagen que te gusta y adaptarla para que se vea bien en tu Kindle o e-reader.

Crear Portadas para Kindle no es una app oficial de Amazon, Kindle, Kobo ni ninguna otra marca de e-readers. Los nombres de marcas se usan únicamente para describir modelos y compatibilidad.

## Market Angle Decision

Market angle selected:
Personalización emocional Kindle-first.

Why this angle should convert in this locale:
En es-MX convierte mejor mostrar que el usuario puede hacer que su Kindle se sienta más suyo con fotos, mascotas, recuerdos o imágenes favoritas. La parte técnica importa, pero debe aparecer después como respaldo.

Primary visual desire:
Ver una imagen querida convertida en una portada terminada para Kindle.

Secondary trust proof:
Recorte, vista previa y selección de modelo para Kindle, Kobo, Nook, PocketBook, Tolino y más.

What should NOT be the first screenshot:
Selector de modelo, pantalla de ajustes, lista técnica, tabla de compatibilidad o una captura que parezca manual de uso.

Feature-first risk:
Si la ficha empieza hablando de tamaño, modelo o compatibilidad, puede parecer una herramienta técnica y no una app para hacer más personal el Kindle.

## Regional Conversion Strategy

Primary user desire:
Hacer que su Kindle se vea más bonito, más personal y más conectado con su vida diaria.

Best use cases to show:
Foto favorita, mascota, arte/aesthetic, recuerdo de viaje, imagen personal adaptada como portada.

What to avoid:
Capturas técnicas al inicio, frases genéricas sobre fondo blanco, promesas de generación de imágenes, tono demasiado frío, o hacer que parezca solo una app de recorte.

Search/ASO terms to include naturally:
portadas para Kindle, crear portada Kindle, portada personalizada, fotos para Kindle, e-reader, Kobo, vista previa, recortar portada, portada aesthetic.

## Notes / Assumptions

- Esta golden ficha enfatiza Kindle-first por ASO, sin ocultar soporte para otros e-readers compatibles.
- La app no debe prometer generación de imágenes, plantillas avanzadas ni creación automática de arte. Adapta imágenes que el usuario ya tiene.
- Los nombres de marcas se usan únicamente de forma descriptiva. No debe implicarse afiliación oficial con Amazon, Kindle, Kobo ni otras marcas.
- La compatibilidad debe describirse como familias/modelos soportados, salvo que el catálogo exacto esté verificado.
- Esta golden ficha es referencia de calidad para Codex, no una plantilla para copiar literalmente a otros locales.
